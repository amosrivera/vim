##Install

`git clone https://amosrivera@github.com/amosrivera/vim.git ~/.vim`<br />
`ln -s ~/.vim/.vimrc ~/`<br />

`cd ~/.vim`<br />
`git submodule init`<br />
`git submodule update`<br />
