##Install

<p>`git clone https://amosrivera@github.com/amosrivera/vim.git ~/.vim`</p>
<p>`ln -s ~/.vim/.vimrc ~/`</p>

<p>`cd ~/.vim`</p>
<p>`git submodule init`</p>
<p>`git submodule update`</p>
